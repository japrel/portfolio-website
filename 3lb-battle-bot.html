Project Overview

The 3LB Battle Bot is a lightweight combat robot built for competitive robot fighting events. It is designed to fit into the 3-pound weight class, while delivering strong performance, speed, and durability.

Key Features

  • Strong chassis and compact design for impact resistance.
  • High-torque drive motors for fast turns and aggressive movement.
  • Responsive remote control using an efficient radio system.
  • Reliable power system with battery management for consistent performance.
  • Simple but effective weapon system designed for the 3LB weight limit.

Technical Description

The robot uses a compact frame built from lightweight materials, with motors placed for balanced traction and maneuverability. The electronics are mounted securely to protect them during battle, and the drive system is tuned for quick acceleration and controlled handling.

This project demonstrates skills in robotics, electronics, mechanical design, and embedded systems. It is a practical example of combining engineering knowledge with real-world competition requirements.
